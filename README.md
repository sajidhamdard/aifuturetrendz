# AI Future Trendz

A static blog about artificial intelligence trends, tools, and insights for developers and tech professionals.

Live site: [aifuturetrendz.com](https://aifuturetrendz.com)

## About

AI Future Trendz covers practical AI topics without the hype. Articles focus on large language models, enterprise AI adoption, generative AI products, and how these technologies affect real-world software and business.

## Topics Covered

- Large language models (LLMs) and how they differ
- ChatGPT, Gemini, and other AI tools
- AI industry trends and market analysis
- How AI is being used in enterprise and software development

## Project Structure

```
├── index.html                  # Home page
├── articles.html               # Articles listing page
├── about.html                  # About page
├── contact.html                # Contact page
├── privacy-policy.html         # Privacy policy
├── terms-and-conditions.html   # Terms and conditions
├── assets/                     # Images and static assets
├── css/                        # Stylesheets
└── scripts/                    # JavaScript files
```

Each article is a standalone HTML file in the root directory.

## Running Locally

This is a static site with no build step required. Open any HTML file directly in a browser, or serve the directory with any static file server.

Using Python:

```bash
python -m http.server 8080
```

Then open `http://localhost:8080` in your browser.
